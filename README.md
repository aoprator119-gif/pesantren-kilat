1. **⏱️ Puasa** - Makna dan Hikmah Puasa
2. **🤲 Kumpulan Doa** - Doa-doa Harian
3. **💰 Zakat Fitrah** - Berbagi Kebahagiaan
4. **🎵 Tajwid** - Cara Membaca Al-Quran
5. **📜 Turunya Al-Quran** - Sejarah Kitab Suci

## 💡 Tips Pengembangan

- Tambahkan sub-materi di folder `materi/`
- Gunakan CSS terpisah di folder `assets/css/`
- Tambahkan gambar di folder `assets/images/`
- Buat halaman admin untuk manajemen konten (advanced)

## 🐛 Troubleshooting

**Halaman tidak muncul?**
- Pastikan file bernama `index.html`
- Tunggu 5-10 menit untuk deploy GitHub Pages
- Cek di Settings → Pages

**Link materi tidak berfungsi?**
- Periksa path file (gunakan `/` bukan `\`)
- Pastikan file materi sudah ada di folder `materi/`
